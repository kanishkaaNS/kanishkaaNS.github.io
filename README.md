# Personal Portfolio Website

## Overview
A modern React-based portfolio website showcasing my frontend projects, technical skills, and development experience. The website is built with performance and clean UI structure in mind.

## Tech Stack
- React
- Vite
- CSS
- JavaScript

## Features
- Smooth scrolling integration
- Reusable component-based architecture
- Responsive design for mobile and desktop
- Optimized asset loading
- Clean and structured project configuration
